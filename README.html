    2. 团队开源项目

    2.1 代码管理

  8. GitHub：全球最大开源代码库
  9. GitLab：持续集成、企业用代码库
  10. SVN：国内传统it企业使用较多
  11. Maven：项目管理及自动构建工具，目前发展如日中天
  12. Gradle：跟Maven类似的，都是项目管理及自动构建工具
  13. Ant：跟Maven类似的，都是项目管理及自动构建工具

    2.2 分布式文件系统

  14. Fastdfs：文件存储，文件同步和文件访问，它可以解决高容量和负载平衡问题。FastDFS满足基于照片共享站点和视频共享站点等文件服务的网站的要求。
  15. Minio：Minio是一款高性能的分布式对象存储服务器，专为大规模私有云基础架构。
  16. Gluster：GlusterFS是一个可扩展的网络文件系统，适用于云存储和媒体流等数据密集型任务。

    2.3 分布式框架

  17. Spring Cloud：目前开源社区最活跃的分布式框架，目前自己也在用。
  18. Dubbo：阿里巴巴的开源分布式框架，致力于提供高性能和透明化的RPC远程服务调用方案，是阿里巴巴SOA服务化治理方案的核心框架。
  19. Finatra：Twitter基于scala开发的分布式框架。
  20. Thrift：Facebook开源的跨语言分布式框架。
  21. Dubbox：当当根据自身的需求，为Dubbo实现了一些新的功能，并将其命名为Dubbox。
  22. Motan：新浪用于快速开发高性能分布式服务的跨语言远程过程调用（RPC）框架。
  23. Grpc：谷歌开源，支持跨语言。
  24. Istio：Google、IBM、Lyft共同开发的新一代Service Mesh开源项目，目前备受关注。

    2.4 权限安全相关

  25. Shiro：ApacheShiro专注于易用性，可以依赖安全，稳定的身份验证，授权，加密和会话管理。
  26. Spring Security：SpringSecurity是一个功能强大且可高度自定义的身份验证和访问控制框架。它是保护基于Spring的应用程序的事实上的标准。
  27. OAuth2：一种安全的授权框架，提供了一套详细的授权机制。用户或应用可以通过公开的或私有的设置，授权第三方应用访问特定资源。
  28. OpenID Connect：OIDC=(Identity, Authentication) + OAuth 2.0。它在OAuth2上构建了一个身份层，是一个基于OAuth2协议的身份认证标准协议。
  29. JWT： 提供了一种用于发布接入令牌（Access Token),并对发布的签名接入令牌进行验证的方法。

    2.5 模板引擎

  30. Freemarker：国内目前用的比较多的老牌模板引擎。
  31. Thymeleaf：SpringBoot开发推荐的HTML模板引擎。

    2.6 持久层框架

  32. Mybatis：支持定制化SQL、存储过程以及高级映射。适合个人开发，上手快。
  33. Hibernate：老牌强大的持久层框架，学习成本相对较高，数据库移植性良好。
  34. Spring Data JPA：Hibernate二次封装，开发效率高。
  35. Spring Data Mongo: 类似spring data jpa。

    2.7 日志框架

  36. Commons-Logging：jdk自带。
  37. Slf4j: 日志接口，通过各种桥接包判断实现，在编译时静态绑定真正的日志库。
  38. Log4j2: Log4j的升级，配置更强大，支持插件化，使用Disruptor实现异步日志，性能最好。

    2.8 数据采集管道

  39. Logstash：ElasticSearch下的数据采集管道，常用于日志分析。
  40. Actuator：SpringBoot的数据收集器，收集指标，了解流量或数据库的状态。

    2.9 搜索引擎

  41. Lucene：一个全文检索引擎的架构，提供了完整的查询引擎和索引引擎，部分文本分析引擎
  42. ElasticSearch：基于Lucene RESTful web接口的搜索服务器，目前最火的搜索引擎开源项目，毕业论文研究的就是这个。
  43. Solr：高性能，基于Lucene的全文搜索服务器，经常与ElaticSearch比较，上手相对于ElaticSearch快。

    2.10 爬虫

  44. Heritrix：最出色之处在于它良好的可扩展性，方便用户实现自己的抓取逻辑。
  45. jsoup：一款Java的HTML解析器，可直接解析URL地址、HTML文本内容。
  46. crawler4j：提供了简单易用的接口，可以在几分钟内创建一个多线程网络爬虫。
  47. WebCollector：一个无须配置、便于二次开发爬虫内核，它提供精简的的API，只需少量代码即可实现一个功能强大的爬虫。
  48. WebMagic：完全模块化的设计,支持多线程抓取，分布式抓取，并支持自动重试、自定义UA/cookie等功能。

    2.11 消息中间件

  49. kafka：scala编写的高吞吐量的分布式发布订阅消息系统，目前较为主流。
  50. RabbitMQ：Erlang编写，在AMQP基础上完整的，可复用的企业消息系统。
  51. RocketMQ：阿里巴巴的MQ中间件,在其多个产品下使用，并能够撑住双十一的大流量。
  52. ActiveMQ：完全支持JMS1.1和J2EE 1.4规范的消息队列。
  53. Apache Pulsar：雅虎开发的企业级的发布订阅消息系统。
  54. Apache DistributedLog：Twitter开发的一个低延时（毫秒级）、高吞吐的分布式复制日志流系统。

    2.12 响应式编程

  55. Reactor：spring社区开源，在spring 5.0直接引入。
  56. Vert.x：基于netty,可以通过它使用JavaScript、Ruby、Groovy、Java、甚至是混合语言来编写应用。
  57. RxJava：安卓上用的特别多，特别与RxAndroid,Retrofit等结合使用。

    2.13 热加载

  58. JRebel：收费，IDEA，Eclipse，Netbeans都有它的插件，目前自己在IDEA上用破解版。
  59. Spring-Loaded：Spring开源热更新工具。
  60. HotswapAgent：免费开源热更新。

    2.14 代码测试

  61. Spock：吸取了JUnit，jMock，RSpec，Groovy，Scala，Vulcans和其他的优秀测试框架，目前自己也在写。
  62. JUnit5：最常使用的单元测试工具。
  63. Mockito：允许使用自动化的单元测试创建和测试双对象，以达到测试驱动开发和行为驱动开发的目的。
  64. JaCoCo：Java代码覆盖库，用Sonarqube测试代码覆盖率需要用到。

    2.15 测试工具

  65. Jmeter：老牌压测工具，也是用的最广泛的。
  66. Fio：文件系统性能测试。
  67. Ab：Ab压测，命令行压测工具。
  68. Pact：提供对消费者驱动的契约测试的支持，支持多种语言。
  69. Spring Cloud Contract：一个消费者驱动的微服务契约测试套件。

    2.16 监控相关

  70. Zipkin：分布式服务监控，可以监控调用链。
  71. Micrometer：Micrometer为最流行的监控系统提供了仪表客户端的简单外观，需要数据管道。
  72. Grafana：开源数据可视化仪表视图服务。
  73. kibana：ElasticSearch下的数据可视化服务。
  74. thingsboard：开源可视化平台。

    2.17 前端相关

  75. Bootstrap：扁平化风格前端UI框架。
  76. AntDesign：阿里出品，在React上的企业前端UI框架。
  77. React：Fackbook出品的前端框架，实现前后端分离开发，前端作为单独服务在Node.js环境下运行，由npm管理包，可以说是目前最流行的前端框架，自己也稍微写过一点。
  78. ReactNative：FaceBook出品的前端移动端前端框架，实现模块化开发，Android开发的模块可以用在IOS上，挺方便的，目前用的比较好的是京东APP。
  79. Vue：渐进式JavaScript 框架。
  80. Angular：Angular使开发人员能够构建在Web，移动设备或桌面上的应用程序。
  81. G6：阿里蚂蚁金服出品的JavaScript中的图形可视化框架。
  82. lodash：前端JavaScript工具库，提供多种数组、字符串处理工具。
  83. Mobx：Mobx 是一个功能强大，上手非常容易的状态管理工具。
  84. Redux：也是一个状态管理工具，比Mobx更加严谨，但是上手更难。
  85. jsTree：jsTree是一个基于jQuery的Tree控件。支持XML，JSON，Html三种数据源。提供创建，重命名，移动，删除，拖放节点操作。
  86. ckeditor：富文本编辑器。
  87. Echarts：百度出品的可视化工具，目前用的较多，现在有React版本的Echarts。
  88. Quill：也是一款富文本编辑器，大多用来做二次开发。

    2.18 分布式相关工具

  89. Eureka：Netflix开源的，java开发，基于restful，用于服务注册。
  90. Consul：Consul是一种分布式，高可用性和数据中心感知解决方案，用于跨动态分布式基础架构连接和配置应用程序。
  91. Ribbon：Netflix开源的，用于负载均衡。
  92. Hystrix：Hystrix是一个延迟和容错库，旨在隔离对远程系统，服务和第三方库的访问点，停止级联故障，并在复杂的分布式系统中实现弹性，在这些系统中，故障是不可避免的。
  93. Zookeeper：非常强大的开源项目，目前主要用途有分布式服务的分布式锁、命名服务、配置管理、集群管理 、队列管理，kafka就是基于Zookeeper实现的消息中间件。
  94. Apollo：携程框架部门研发的分布式配置中心。
  95. Kong：是一个云原生，快速，可扩展和分布式微服务抽象层（也称为API网关，API中间件或在某些情况下为Service Mesh）。
  96. Spinnaker：是一个开源的，多云连续交付平台，用于以高速度和可靠性发布软件更改。
  97. Nginx：是一个异步框架的Web服务器，也可以用作反向代理，负载平衡器 和HTTP缓存。

    2.19 数据库相关

  98. Mysql：适用于个人的小型开源数据库，现在微服务比较热门，一个大型服务拆分为模块服务，Mysql就比较适用于微服务。
  99. Oracle：老牌数据库，非常强大，现在收购了Sun公司。
  100. MogoDB：非关系型的数据库的典型代表。
  101. HBase：是一个开源的非关系型分布式数据库，大数据存储。
  102. Liquibase：一个用于跟踪、管理和应用数据库变化的开源的数据库重构工具。支持xml，groovy编写。
  103. Flyway：是一个敏捷工具，用于数据库的移植。
  104. Apache Impala：是一个高性能分析数据库，可针对存储在 Apache Hadoop集群中的PB级数据进行闪电般快速的分布式 SQL查询。
  105. MyFlash：美团点评的开源MySQL闪回工具。
  106. H2 DataBase：内存数据库，可以用于单元测试中的数据库。
  107. Ctrip DAL：是携程框架部开发的数据库访问框架，支持代码生成和水平扩展。
  108. HikariCP：性能最高的数据库连接池。
  109. Druid：阿里出品，为监控而生的数据库连接池。
  110. Phoenix：构建在HBase上的一个SQL层，能让我们用标准的JDBC APIs而不是HBase客户端APIs来创建表，插入数据和对HBase数据进行查询。

    2.20 容器相关

  111. Docker：是一个开源的应用容器引擎，目前主流容器技术，虚拟机的最大竞争对手。
  112. Kubernetes：谷歌开源用于自动化部署，扩展和管理集装箱化应用程序的系统，也是目前容器编排主流开源项目。
  113. Container-Diff：Google 开源 Docker镜像差异分析工具。
  114. clair：容器的漏洞静态分析。
  115. Helm：预先配置的Kubernetes资源包管理工具，目前我们用Helm管理容器环境变量等配置。
  116. Harbor：Docker镜像仓库管理平台，也是主流的容器相关平台。
  117. Openshift：红帽公司出品的一个功能全面容器应用平台，底层管理也是使用Kubernetes的。

    2.21 编程语言

  118. Groovy：类似Ruby的动态语言，Gradle的编写语言。
  119. Scala：强大的多范式编程语言，有Spark，Akka，Kafka等开源项目。
  120. Clojure：动态Lisp方言。
  121. Kotlin：由 JetBrains 设计开发并开源，多用于安卓开发，简化版的Scala，是谷歌现在力推的安卓官方开发语言，前景光明。
  122. Lux：正在开发中的静态Lisp方言。
  123. Swift：苹果公司用来取代Objective-C的语言，用于开发Mac OS和iOS，十分热门。
  124. GO：是Google开发的一种静态强类型、编辑型、并发型，并具有垃圾回收功能的编程语言。
  125. Python：是一种解释型、面向对象、动态数据类型的高级程序设计语言，多用于数据分析等。
  126. R：R语言，一种自由软件编程语言与操作环境，主要用于统计分析、绘图、数据挖掘。

    2.22 工具相关

  127. Guava：Google开源，十分强大，包含许多Google核心的Java常用库，Java8吸取了Guava的许多优点发展自己的新特性。
  128. Apache Commons：包含了很多开源的工具。
  129. Lombok：注解通过编译器生成getter,setter,builder等代码。
  130. Chronicle Map：基于内存的键值对存储,低延迟、高并发,支持持久化到磁盘，以及多键值查询。
  131. reflections：一个java反射工具包。
  132. strman-java：一个字符串处理工具,使用它可以解决几乎所有字符串处理场景。
  133. Jgit: java实现git操作。
  134. Redisson：在Redis的基础上实现的Java驻内存数据网格（In-Memory Data GridEhcache – Java分布式缓存框架）。它不仅提供了一系列的分布式的Java常用对象，还提供了许多分布式服务，可以用于分布式服务锁。
  135. OpenGrok：Oracle公司的，用Java编写的源代码库搜索工具。
  136. mybatis-3：Mybatis在Java的映射工具包，源码设计缓存设计、大量的反射，挺值得看的。

    2.23 缓存框架

  137. Ehcache：Java分布式缓存框架。
  138. Cacheonix：高性能Java分布式缓存系统。
  139. ASimpleCache：轻量级Android缓存框架。
  140. JBoss Cache：基于事物的Java缓存框架。
  141. Voldemort：基于键-值（key-value）的缓存框架。
  142. Redis：应用最多的缓存框架，常用于二级缓存，不仅可以作为分布式服务的分布式锁，还可以用做pubsub实现分布式WebSocket。

    2.24 架构相关

  143. Domain-driven design：领域驱动设计，复杂的业务逻辑可以参考，其中盒马生鲜就用到了其中的概念。
  144. Saga：用于解决分布式系统事务一致性的模式。

    2.25 工作流引擎

  145. Activiti：目前用的比较多的工作流，重点关注在系统开发的易用性和轻量性上。
  146. JBPM：是JBoss中一款开源的工作流引擎，是一个轻量级的，使用最新的BPMN 2规范可扩展的，目前最新的是JBPM3。

    2.26 代码标准

  147. p3c：阿里巴巴代码规约。
  148. SonarQube：一个开源的代码质量管理系统。

    2.27 其他

  149. PArallel：百度深度学习框架。
  150. netty：事件驱动的异步网络应用程序框架。
  151. Tensorflow：谷歌机器学习框架，目前最流行的机器学习框架。
  152. mpvue：美团小程序开发框架。
  153. Choerodon：自己参与开发的开源企业服务平台，是基于Kubernetes的容器编排和管理能力，整合DevOps工具链、微服务和移动应用框架。
  154. Pinpoint：是一个开源的APM（应用程序性能管理）工具，适用于用Java编写的大型分布式系统。

    3. 个人优秀开源项目

  160. translations：oldratlee在外文中关于一些知识的中文总结，值得一看。

  169. KnowledgeManage：个人前端技能树，友情链接。

    3.2 框架、工具

  170. jodd：轻量级的Java微框架。
  171. tini：容器的一个小而有效的init。
  172. github-markdown-toc：用Shell写的自动生成Github中Markdown文档目录的小工具。
  173. undertow：简易高性能无阻塞网络Java服务器框架。
  174. Lealone：兼具RDBMS、NoSQL优点的个人异步化NewSQL单机与分布式关系数据库。
